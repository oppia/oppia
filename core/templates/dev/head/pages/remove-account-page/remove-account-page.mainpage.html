Remove account

Overview

This action will remove this user account and also all the private data that are associated with this account. Already published data will be anonymized, so they can't be associated with this account and the ownership of that data will be transferred to the community. Summary of data tha will be removed:

  • User and email settings
  • User statistics
  • Private explorations, collections, topics, stories, skills, questions

Summary of data that will be anonymized:

  • Threads and suggestions
  • User statistics
  • Public explorations, collections, topics, stories, skills, questions

List of deleted data models

  • UserSubscriptionsModel
  • GeneralFeedbackEmailReplyToIdModel
  • ExplorationUserDataModel
  • UserSettingsModel
  • UserEmailPreferencesModel

List of anonymized data models

  • ExplorationUserDataModel
  • GeneralFeedbackThreadModel
  • GeneralFeedbackMessageModel
  • GeneralSuggestionModel
