Update Translations

It looks like you are updating content for a published curated lesson that contains translations in different languages.


Please select an appropriate change for the translations:

  • Remove existing translations: Remove all translations for the given content. Select this if your changes are significant and the existing translations aren't appropriate for the updated content.

  • Mark translations as stale: Mark the existing translations for the given content as stale. Select this if your changes will require the translations to be updated, but the previous translations can still be used.

  • Leave as is: Leave all translations as they are. Select this if your changes do not materially affect translations, e.g. if you fixed a minor typo in the source text.

Note:The translation updates will only be applied when these changes are saved.
